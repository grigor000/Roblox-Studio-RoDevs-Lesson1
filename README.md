Building Client script handles the visual part, such as showing the placement position and allowing the player to change shape, size, rotation, and other properties.

Building Server script handles all important validations, such as checking whether the part is too far from the player and preventing any possible exploitation.

Building Module script receives data from the server and processes all placement parameters for creating the object.
